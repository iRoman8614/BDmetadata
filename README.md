# BDmetadata
RESTful API на Node.js, которое служить в качестве интерфейса для управления метаданными моделей, хранящихся в базе данных.

После скачивания репозитория выполните следующие действия
1. В терминале перейдите в папку проекта и выполните программу npm install

2. В корневой папке создайте файл .env

заполните его следующим образом, чтобы связать с вашей базой данных
PORT=____
DB_NAME=____
DB_USER=____
DB_PASSWORD=____
DB_HOST=____
DB_PORT=____

3. запустите скрипт node run dev

4. Можно приступить к тестированию запросов через программы postman или insomnia
